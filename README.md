##🚀 How to Run

```1. Create project (already done)
flutter create portfolio_app

 2. Add dependencies
flutter pub add material3 animate_do flutter_animate google_fonts shimmer

3. Add assets
 - assets/images/avatar.png (your photo)
 - assets/fonts/Inter-Regular.ttf
- assets/fonts/Inter-Bold.ttf

4. Run on Android
flutter run

5. Run on iOS
flutter run -d macos

6. Build APK
flutter build apk --release

7. Build for web
flutter build web
